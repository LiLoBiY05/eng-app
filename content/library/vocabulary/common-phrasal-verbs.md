# Common Phrasal Verbs — Detailed Guide

A detailed breakdown of 19 high-frequency phrasal verbs used in everyday English.

## Detailed Breakdowns

### 1. KEEP UP WITH

| Meaning | Example | Translation |
|---------|---------|-------------|
| Встигати за (рухатись з однаковою швидкістю) | I can't keep up with you — slow down! | Я не встигаю за тобою — сповільни! |
| Не відставати (підтримувати темп) | He keeps up with the latest trends. | Він стежить за останніми трендами. |
| Бути в курсі | Do you keep up with the news? | Ти стежиш за новинами? |
| Підтримувати контакт | We keep up with old friends online. | Ми підтримуємо зв'язок зі старими друзями онлайн. |

---

### 2. CATCH UP WITH

| Meaning | Example | Translation |
|---------|---------|-------------|
| Наздогнати | Run faster to catch up with them! | Біжи швидше, щоб їх наздогнати! |
| Зустрітися / поспілкуватися | Let's catch up with Sarah tonight. | Давай зустрінемося з Сарою сьогодні ввечері. |
| Наздогнати (роботу/навчання) | I need to catch up with my work. | Мені треба наздогнати свою роботу. |
| Застукати (поліція) | The police caught up with him. | Поліція його спіймала. |

> **Keep up with** = не відставати постійно (continuous action)
> **Catch up with** = наздогнати після відставання (one-time action)

---

### 3. CHECK OUT

| Meaning | Example | Translation |
|---------|---------|-------------|
| Виписатися з готелю | We check out at 11 AM tomorrow. | Ми виписуємося завтра о 11 ранку. |
| Перевірити / подивитися | Check out this amazing video! | Подивись це дивовижне відео! |
| Взяти в бібліотеці | I checked out three books. | Я взяв три книги. |
| Оплатити на касі | I'll check out these items. | Я оплачу ці речі. |

⚠️ Always check the context — **check out** has many meanings!

### 4. MAKE FOR

|---------|---------|-------------|

| Сприяти / призводити до | Good communication makes for success. | Добра комунікація сприяє успіху. |

---

### 5. SET OUT / SET OFF

| Meaning | Example | Translation |
|---------|---------|-------------|
| Вирушати в дорогу | We set out at dawn. | Ми вирушили на світанку. |
| Розпочинати щось | He set out to learn Spanish. | Він розпочав вивчення іспанської. |
| Викладати ідеї (set out only) | He set out his ideas clearly. | Він чітко виклав свої ідеї. |
| Спричинити / активувати (set off only) | The smoke set off the alarm. | Дим спрацював сигналізацію. |

> **Set out** = більш формально; для викладання ідей — тільки **set out**
> **Set off** = менш формально; для спрацювання сигналізації — тільки **set off**

### 18. MAKE UP

| Meaning | Example | Translation |
|---------|---------|-------------|
| Вигадувати | He made up an excuse. | Він вигадав виправдання. |
| Миритися після сварки | They made up after the argument. | Вони помирилися після сварки. |
| Складати / становити | Women make up 60% of the class. | Жінки становлять 60% класу. |
| Накладати макіяж | She's making up for the party. | Вона наносить макіяж на вечірку. |

---

### 19. MAKE UP FOR

| Meaning | Example | Translation |
|---------|---------|-------------|
| Компенсувати | I'll make up for my mistake. | Я компенсую свою помилку. |
| Надолужувати | He's trying to make up for lost time. | Він намагається надолужити втрачений час. |

> **Make up** = миритися, вигадувати, складати (різні значення)
> **Make up for** = компенсувати щось конкретне (always with FOR)

## Common Mistakes

| ❌ Wrong | ✅ Correct | Note |
|---------|-----------|------|
| I catch up with the news every day. | I **keep up with** the news every day. | Постійно стежити = KEEP UP WITH |
| I can't keep up the pace. | I can't **keep up with** the pace. | Потрібен WITH |
| He set off his plans clearly. | He **set out** his plans clearly. | Викладати ідеї = SET OUT |
| Drop off me at the corner. | **Drop me off** at the corner. | Займенник між дієсловом і частинкою |
| We set out the alarm. | We **set off** the alarm. | Спрацювати = SET OFF |
